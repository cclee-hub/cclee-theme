# 外贸焊接工具网站设计方案

**日期：** 2026-03-20
**项目：** Yougu WordPress 外贸网站

---

## 项目概述

| 项目 | 内容 |
|------|------|
| 类型 | 外贸企业官网 + WooCommerce 商城（混合型） |
| 产品 | 焊接工具（电烙铁、焊台、烙铁嘴） |
| 目标市场 | 全球市场 |
| 语言 | 英文 |
| 货币 | USD |
| 主题 | Astra Child |

## 业务模式

- **B2C：** 标准产品直接在线购买
- **B2B：** 大额/定制需求走询盘

## 网站结构

### 导航栏
```
Logo | Home | About Us | Products | Contact | Search | Cart
```

### 页面清单

| 页面 | 说明 |
|------|------|
| Home | 品牌展示、产品亮点、公司简介、CTA |
| About Us | 公司介绍、数字亮点、工厂展示 |
| Products | WooCommerce 商店页（产品分类 + 列表） |
| Contact | 联系方式、询盘表单 |

---

## 首页设计

### 模块顺序（从上到下）

1. **Hero Banner** - 主视觉 + 标语 + CTA（Shop Now / Contact Us）
2. **产品分类** - 图标/图片网格：Soldering Irons / Stations / Tips
3. **热销产品** - WooCommerce 产品轮播（4-6个）
4. **公司简介** - 简短介绍 + 工厂图片 + 数字亮点
5. **客户评价** - 2-3 条 testimonial 卡片
6. **品牌 Logo** - 合作伙伴/认证 Logo 轮播（后期补充）
7. **CTA 条** - "Get a Quote Today" + 按钮
8. **页脚** - 联系信息 + 快捷链接 + 社交图标

- 触发：点击 "Inquiry" 按钮
- 字段：姓名*、邮箱*、公司、国家(下拉)*、消息*
- 功能：自动填入产品名称，提交后发送邮件
